### NyaNeko : A Web For Meow Lovers

The reason i built this was for the FOW project submission at my school. This project uses simple html, tailwind css and vanilla javascript and an api which gives us some cool facts about cats !! 

## Key Features - 

- Uses plain html, tailwind css and vanilla javascript 
- Uses meowfact api : https://github.com/wh-iterabb-it/meowfacts.git
- Features clean and minimal looking UI
- Responsive to various devices including mobile phone

## How to view the project? 

Click on the link which will redirect you to the page hosted through vercel : https://nya-neko.vercel.app/
